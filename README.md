# VSOP_FX
Libraries for the Mitaka VSOP-FX Correlation Center

This repository consists of three sections:
- FX
  This contains quality assuarance tools for VLBI correlation data.

- FXLOG
  Interfaces for observation log files to feed into correlation operations

- FXSYNC
  Control software for synchronized plabacks

These libraries were build under fcc (Fujitsu C compiler) for Solaris.
